# Vrinda Store Annual Sales Dashboard

This project presents an interactive Excel dashboard built using Microsoft Excel to analyze annual sales performance.

## Features

* Dynamic KPI showing Total Sales
* Interactive slicers (Month, Category, Channel, Gender, State)
* Top 5 performing states analysis
* Sales comparison between men and women
* Orders vs Sales visualization

## Tools Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Data Visualization
* Dashboard Design
